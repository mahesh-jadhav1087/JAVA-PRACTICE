# Java Map Collection Practice ☕

This repository contains Java programs and practice problems based on the **Map Collection Framework**. The goal is to build a strong understanding of key-value data structures, internal working of HashMap, and real-world applications using Java Collections.

## 📘 Topics Covered

### Map Basics
- Introduction to Map
- Key-Value Pair Concept
- Difference between List, Set, and Map
- Common Map methods

### HashMap
- Add elements using `put()`
- Retrieve data using `get()`
- Update and remove values
- `containsKey()` and `containsValue()`
- `size()`, `clear()`, `isEmpty()`

### HashMap Iteration
- `keySet()`
- `values()`
- `entrySet()`
- `Map.Entry`

### Internal Working of HashMap
- Hashing
- Bucket concept
- HashCode generation
- Collision handling
- Linked List and Red-Black Tree
- Load Factor
- Initial Capacity
- Rehashing

### LinkedHashMap
- Insertion order
- Internal structure
- Practice examples

### TreeMap
- Sorted key storage
- Natural ordering
- Red-Black Tree concept

### Hashtable
- Thread safety
- Difference between HashMap and Hashtable
