Vector Collection Practice in Java
---
This repository contains Java practice programs based on the Vector Collection Framework and POJO Classes.
The programs cover basic to intermediate operations using Vector, along with logical operations on custom objects like Student, Employee, Product, Player, Book, Movie, etc.
---

**Topics Covered**


Creating and using Vector
Adding elements using add()
Displaying elements using loops
Searching elements
Removing elements
Updating elements
Finding maximum value
Counting elements based on conditions
Copying vectors using addAll()
Using firstElement() and lastElement()
Working with POJO classes
Logical filtering using conditions
Object storage using Vector<Objects>
