# Set Collection in Java

This repository contains Java programs based on the Set Collection Framework.

The programs are created to practice core concepts of Set implementations such as HashSet, LinkedHashSet, and TreeSet along with important interview-based problems.

---

## 📚 Topics Covered

- HashSet
- LinkedHashSet
- TreeSet
- Removing Duplicates
- Set Operations
- Sorting using TreeSet
- Unique Element Problems
- Custom Objects in HashSet
- hashCode() and equals() methods
