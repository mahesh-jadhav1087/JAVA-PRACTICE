# 📘 Core Array Problems in Java

This folder contains important **array-based problems** implemented using **pure array logic (no collections or advanced libraries)**.
The goal is to build strong fundamentals for coding interviews and problem-solving.

---

## 🚀 Problems Covered

### 🔹 1. Find Duplicate Elements

Identify elements that appear more than once in the array.

---

### 🔹 2. Remove Duplicates from Array

Return only unique elements using a temporary array.

---

### 🔹 3. Find Maximum & Minimum

Find the largest and smallest element in a single traversal.

---

### 🔹 4. Rotate Array (Left Rotation)

Shift array elements to the left by a given number of positions.

---

### 🔹 5. Move Zeros to End

Move all zero elements to the end while maintaining order of non-zero elements.

---

### 🔹 6. Count Frequency of Elements

Count how many times each element appears in the array.

---

### 🔹 7. Check if Array is Sorted

Verify whether the array is in ascending order.

or more

---

### 🔹 8. Union & Intersection of Two Arrays

* **Union** → All unique elements from both arrays
* **Intersection** → Common elements between arrays

---

### 🔹 9. Merge Two Sorted Arrays

Merge two sorted arrays into a single sorted array using two-pointer technique.

## ⚙️ Approach Style

* All problems are solved using **basic array operations**
* No use of:

  * HashSet
  * HashMap
  * Collections Framework

This helps in understanding the **core logic deeply** before moving to optimized solutions.
